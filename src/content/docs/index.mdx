**ZXW Graphboard** is an online graph editor for [ZXW calculus](https://en.wikipedia.org/wiki/ZX-calculus),
a diagrammatic calculus used in quantum computing. Place vertices, connect them with edges,
assign phase labels, and export the resulting graph as JSON — all in the browser, with no
account or backend.

Try it live at **[zxwgraphboard.netlify.app](https://zxwgraphboard.netlify.app)**, or browse the
[source code on GitHub](https://github.com/harryhanYuhao/graphboard).

## What you can do

- Draw ZXW diagrams with eight generator types (Z/X spiders and boxes, H, W, And gate, empty node)
- Connect vertices with click-to-connect edges, including fan-out from multiple sources
- Annotate spiders and boxes with phase expressions (`\pi/2`, `3\alpha + 1`, …) rendered with KaTeX
- Rotate vertices, undo/redo, copy/paste subgraphs
- Autosave to your browser; export and import graphs as JSON files
